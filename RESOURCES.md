# Resources

A curated collection of resources for learning about Nix, Home Manager, and flakes.

## Nix & Home Manager Documentation

- [Home Manager Repository](https://github.com/nix-community/home-manager) - Official Home Manager repository with source code and documentation
- [Home Manager Release Notes](https://nix-community.github.io/home-manager/release-notes.xhtml) - Release notes and highlights for Home Manager versions

## Package & Option Search

- [NixOS Package Search](https://search.nixos.org/packages) - Search for Nix packages across channels
- [Home Manager Options Search](https://nix-community.github.io/home-manager/options.xhtml) - Search for Home Manager configuration options and program settings

## Flake Configuration

- [NixOS Templates](https://github.com/NixOS/templates) - Official Nix flake templates for various use cases
- [Home Manager Flake Config](https://www.chrisportela.com/posts/home-manager-flake/) - Guide to setting up Home Manager with flakes
- [Nix Flakes Blog Post](https://gvolpe.com/blog/nix-flakes/) - Introduction to Nix flakes and their benefits

## Containerization (Podman)

- [Rootless Podman Home Manager](https://github.com/FabianKoehnen/rootless-podman-homemanager) - Example configuration for rootless Podman with Home Manager
- [Reddit Discussion on Podman](https://www.reddit.com/r/NixOS/comments/1bh7vy8/does_anyone_have_a_working_example_of_a/) - Community discussion about Podman setup

## Advanced Topics

- [Home Manager DAG Module](https://github.com/nix-community/home-manager/blob/master/modules/lib/dag.nix) - Directed acyclic graph implementation for activation scripts
- [Home Manager Files Module](https://github.com/nix-community/home-manager/blob/b8b391ad18b2ec47d1e95e247301edd4e8326154/modules/files.nix#L50-L83) - File management and activation script implementation
